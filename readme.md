# ETL com Python

Projeto desenvolvido durante bootcamp da DIO com foco em ETL (Extract, Transform, Load).

## Tecnologias utilizadas
- Python
- Pandas

## Fluxo ETL

### Extract
Criação de dados fictícios de usuários.

### Transform
Geração de mensagens personalizadas para cada usuário.

### Load
Exportação dos dados processados para CSV.

## Como executar

```bash
pip install pandas
python etl.py
